# Sensitive Data × LLMs — Handbook

A reference handbook for building AI interfaces over sensitive data: mental models, threat catalog, architecture patterns, decision matrix, implementation playbook.

Source for both the web site (Astro Starlight, deployed to GitHub Pages) and a printable PDF.

## Develop

```bash
nvm use 22          # or any Node ≥ 22
npm install
npm run dev         # http://localhost:4321
```

## Build

```bash
npm run build       # static site → dist/
npm run build:pdf   # static site + handbook.pdf (requires playwright)
```

For PDF generation, first time only:

```bash
npm i -D playwright
npx playwright install chromium
```

## Add or edit content

All content lives in `src/content/docs/`. Stub files are generated by:

```bash
npm run scaffold    # idempotent; won't overwrite existing files
```

Sidebar order is controlled by the `sidebar.order` frontmatter field on each page.

## Deploy

Pushing to `main` triggers `.github/workflows/deploy.yml`, which builds the site and publishes it to GitHub Pages.
